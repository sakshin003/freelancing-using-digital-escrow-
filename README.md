# freelancing-using-digital-escrow-
Freelancing using Digital Escrow is a decentralized web application that enables secure transactions between clients and freelancers. It integrates blockchain technology and MetaMask to hold payments in escrow via smart contracts, releasing funds only when work is verified and completed.

# Clone the repository
git clone https://github.com/sakshin003/freelancing-digital-escrow.git

# Navigate to project folder
cd WhizPlatform-main

# Install dependencies
npm install

# Start the development server
npm start
